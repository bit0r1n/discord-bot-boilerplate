## Discord bot boilerplate
cool boilerplate discord bot with eris !

### How to use
 * Clone this repo `git clone https://github.com/bit0r1n/discord-bot-boilerplate`
 * Run `npm i`
 * Modify how you want
 * Run `npm run build`
 * Run `npm start`
 * gg
